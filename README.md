# Team Formation Portal

A web application for managing student team formations. This application allows students to create teams, view existing teams, and edit team details with a secret key.

## Features

- Create teams with 3-4 members
- Auto-fill student details using last 3 digits of USN
- Prevent duplicate team members
- Generate automatic team numbers (TEAM1, TEAM2, etc.)
- Secret key protection for team editing
- View all formed teams
- Responsive design
